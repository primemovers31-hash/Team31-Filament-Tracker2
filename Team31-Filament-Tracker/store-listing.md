# Google Play Store Listing - Team31 Filament Tracker

## App Name

Team31 Filament Tracker

## Short Description

Track filament spools, scan NFC tags, and manage printer inventory.

## Full Description

Team31 Filament Tracker helps you manage filament inventory across your printers. Use the Android app to scan spool NFC tags and the web app to review inventory and print labels.
